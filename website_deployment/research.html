Research

Datasets

Four datasets were used and joined to create the data used for visualizations and machine learning:

1. 2006 - 2012 School Demographics and Accountability Snapshot, which contains demographic information regarding NYC cohorts and schools (referred to as "Demographic_df")

2. 2016-2017 Graduation Outcomes School, which contains information regarding NYC schools' success metrics (referred to as "Graduation_df")

3. 2017-18 - 2021-22 Demographic Snapshot, which contains additional demographic information for NYC schools, notable figures on the percentage of students living in poverty and the Economic Need Index (referred to as "Poverty_df")

4. Average SAT Scores for NYC Public Schools, which contains success metrics as well as latitude and longitude (for visualizations), referred to as "Latlng_df"


Extract, Load, Transform

The four datasets above contained information regarding cohorts over multiple years and included different age groups of students. Our goal was to correlate demographic information and high school success metrics. In order to compare the information school by school, we took the following steps:

1. Reduced the datasets to include only schools that had students in 9th, 10th, 11th, and/or 12th grades

2. Cleaned the data, including: dropping unnecessary columns, converting non-numeric datatypes to numeric datatypes, and using regular expressions where needed to remove unnecessary non-numeric characters

3. For each dataset, using groupby in Pandas based on the DBN (an identifying code unique to each school) to average all the schools' rows of information on a given feature, thus creating new "averaged" dataframes with one row per school

4. Joining these averaged dataframes together on the DBN column to create a larger, merged dataframe that contains all needed demographic and success metric information


The averaged dataframes based on the first three datasets above were joined in pgAdmin and stored using Amazon Web Services for the machine learning portion of our work. The averaged dataframes using all four of the datasets above were joined using Python/Pandas in a Jupyter Notebook, exported as a CSV, and visualized in Tableau. The reason for the discrepancy in dataframes is that the fourth dataset ("Latlng_df") was used only for latitude and longitude to create map visualizations in Tableau and was not needed for the machine learning work.

Please visit Machine Learning for more information on our machine learning models, Visualizations for the Tableau visualizations of the datasets averaged by school, and our GitHub repository for the code containing our extracting and transforming steps.
